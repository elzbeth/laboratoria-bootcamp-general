CONTANDO CARACTERES Y ESPACIOS

Crea una web que pida, por medio de un prompt(), una frase al usuario. La web debe presentar un mensaje que diga: el número de caracteres y de espacios que componen el string ingresado. En la consola, debe aparecer cada caracter impreso en una línea separada, en mayúsculas. Si el usuario ingresa un cambpo vacio, la web debe dar un mensaje de error.

Arma tu archivo para que tenga 3 funciones:

  • Una función que imprima todos los caracteres de un string, cada uno en una linea separada, en mayúsculas.
  • Una función que cuente el número de caracteres de un string (¡sin utilizar .length!)
  • Una función que cuente el número de espacios de un string
