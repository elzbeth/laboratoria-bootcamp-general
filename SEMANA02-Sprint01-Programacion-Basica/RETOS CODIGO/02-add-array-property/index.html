RETOS CODIGO SEMANA 01 - ADD ARRAY PROPERTY

Escribe una funci√≥n llamada "addArrayProperty(obj, key, array)".

Dado un objeto, una clave y un array, "addArrayProperty" establece una nueva propiedad sobre el objeto en la clave dada, con su valor establecido en el array dado. Ejemplo de entrada: ```js var myObj = {}; var myArray = [1, 3]; addArrayProperty(myObj, 'myProperty', myArray); console.log(myObj.myProperty); // --> [1, 3] ```
