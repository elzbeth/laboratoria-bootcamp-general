Escriba una función llamada `getNthElement`. Dada una matriz y un entero, `getNthElement` devuelve el elemento al entero dado, dentro de la matriz dada.

Notas: * Si el array tiene una longitud de 0, debería devolver `undefined`.

Ejemplo de entrada:

```js var output = getNthElement([1, 3, 5], 1); console.log(output); // --> 3 ```
